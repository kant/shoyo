# Quick Start

Shoyo can be imported from both npm and CDN

From npm

```bash
yarn add shoyo 
//or
npm i shoyo
```
Import the stylesheet in your code

```js
import "shoyo/dist/main.css";
```

From CDN

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/soulsam480/shoyo/dist/main.css">
```
